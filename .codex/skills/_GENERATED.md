# Generated Codex Skill Stubs

This directory is generated by `sync_codex_stubs.py`. Each stub carries the
canonical skill's name/description frontmatter plus a redirect body; the skill
content itself lives only under `.claude/skills/`.

Do not hand-edit files under `.codex/skills`. Edit `.claude/skills`, then run:

    python3 .claude/skills/ppt-master/scripts/sync_codex_stubs.py
